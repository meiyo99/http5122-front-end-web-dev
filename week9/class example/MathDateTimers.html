Math, Dates & Time

The Big 3 Concepts

  1. JavaScript has a special Math object with built in properties and methods for performing mathematical operations.
  2. JavaScript has a special Date object with properties and methods that let us manage dates and times in our logic.
  3. The browser window object has built in timers that we can use for timing operations.
